# Vue 3 + Vite

這份範本可協助你開始使用 Vue 3 與 Vite 進行開發。

範本採用 Vue 3 的 `<script setup>` 單一檔案元件（SFC）寫法；可參考 [script setup 文件](https://v3.vuejs.org/api/sfc-script-setup.html) 了解更多。

關於 IDE 支援與開發工具，請參考 [Vue 文件的工具化指南](https://vuejs.org/guide/scaling-up/tooling.html)。

> 注意：這是由 Vite 建立專案時產生的基礎說明，不是 v5 網站的設計計劃書或內容規格。
